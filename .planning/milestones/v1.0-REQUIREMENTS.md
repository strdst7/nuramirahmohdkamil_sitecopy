## v1 Requirements (all complete)

### Pages & Navigation

- [x] **PAGE-01**: Global shared layout with header (logo + nav links), footer, and bottom navigation dock connecting all 5 pages
- [x] **PAGE-02**: Home/Landing page — hero section with elevator pitch in massive typography, visual terminal preview, and CTAs
- [x] **PAGE-03**: Insights/Blog page — journal-style article entries with surreal diagrams, asymmetric layout
- [x] **PAGE-04**: AI Playground page — interactive terminal with prime-directive input, sliders, output basin, and live API integration
- [x] **PAGE-05**: Portfolio/Projects page — "drifting islands" project cards with organic blob shapes

### Design System

- [x] **DSGN-01**: Build-time Tailwind config with all Surrealist Echoes design tokens (46 colors, typography, spacing)
- [x] **DSGN-02**: Shared component library — GlassPanel, OrganicCard, DripBorder, PulseButton, MeltingShadow, Icon
- [x] **DSGN-03**: Typography system with self-hosted fonts via next/font (Epilogue, Newsreader, Space Grotesk)
- [x] **DSGN-04**: Responsive layout adapting from mobile (single column) through desktop (asymmetric fluid)

### AI Playground

- [x] **PLAY-01**: OpenAI text generation via proxied POST /api/ai/openai (SSE streaming, gpt-4o-mini)
- [x] **PLAY-02**: Gemini multimodal analysis via proxied POST /api/ai/gemini (gemini-2.0-flash)
- [x] **PLAY-03**: Interactive terminal UI — typewriter input, melting-clock loader, streaming output
- [x] **PLAY-04**: Temperature (0-2.0) and hallucination (0-1.0) sliders with Surrealist Echoes gradient styling
- [x] **PLAY-05**: API key security — all AI calls proxied through server routes, zero client key exposure

### Lead Capture

- [x] **LEAD-01**: Lead capture form — project name, email (optional), WhatsApp (optional), intent selector
- [x] **LEAD-02**: POST /api/leads endpoint saving to Supabase playground_leads table
- [x] **LEAD-03**: Form validation with inline error states styled in Surrealist Echoes aesthetic

### Infrastructure & SEO

- [x] **INFR-01**: Next.js App Router scaffold with TypeScript, PostCSS, and Tailwind build pipeline
- [x] **INFR-02**: Local image assets replacing Google-hosted placeholders, optimized via next/image
- [x] **INFR-03**: SEO metadata — per-page title, description, OG tags, sitemap.xml, robots.txt
- [x] **INFR-04**: Vercel deployment + Docker support with environment variables and analytics
