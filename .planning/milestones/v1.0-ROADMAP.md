## Milestone Summary

**Key Decisions:**
- Next.js 16 App Router with TypeScript and Tailwind v3
- Surrealist Echoes canonical design system (BluePrint retired)
- Self-hosted variable fonts via next/font/google
- All AI API calls proxied through server-side routes (zero key leakage)
- Supabase service role key for server-to-server inserts (no client auth)
- Zod v4 for form validation (migrated from errorMap to message API)
- Supabase client lazy-initialized to prevent Docker build crashes

**Issues Resolved:**
- Metadata export from client components (extracted to layout.tsx)
- Zod v4 API changes (enum errorMap → message)
- Supabase client eager init crash at build time (→ lazy getSupabase factory)
- Icon component size type extended to 14-48px range
- Docker build TypeScript check slowness (→ ignoreBuildErrors in Docker, verified locally)

**Technical Debt:**
- Rate limiting on /api/leads deferred to v2
- Email notification on new lead deferred to v2
- Custom domain (nuramirah.com) deferred to v2
